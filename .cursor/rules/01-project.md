# Project Rules (Landing Lab)

- Objetivo: criar landing pages semanais simples para estudo.
- Stack: Next.js (App Router) + Tailwind + shadcn/ui.
- Usar Sonner para toast, não usar toast antigo deprecated.
- Cada landing deve existir em `src/app/(landings)/week-XX/`.
- Copy e dados sempre em `content.ts` para facilitar edição.
- Priorizar componentes reutilizáveis em `src/components/landing/`.
- Escopo: evitar backend, auth, banco, integrações complexas.
